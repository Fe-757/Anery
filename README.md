# Anery
Anery é uma linguagem de programação simples em português, focado para iniciantes e devs que não querem ter dor de cabeça.
Está em fase bem inicial, pretendo expandir bem mais essa linguagem

## Descrição
Como dito acima, Anery é focado principalmente para quem quer aprender lógica/programação de forma simples, totalmente em português e com lógica simples.

## Como executar
O interpretador do Anery fica no site Anery.rf.gd, hospedado pelo infinityfree então espere avisos de falta de certificados

lógica e sintaxes:

[] - cria funções
{} - representa valores
# - comentários 
. - diz oque deve fazer 
calc - calcular valor numérico
env - enviar valor
rec - receber valor
res - resultado
console.env - exibir no console

exemplo:

[calc {8}+{4}.res
console.env {res}]
